\section{Заключение}
Основные результаты, полученные за время обучения в аспирантуре.
\begin{enumerate}
    \item Предложен подход к проверке асимптотики собственных чисел грамиана управляемости линейных систем с малым параметром, основанный на разложении по степеням малого параметра.
    \item Доказаны достаточные условия выпуклости множеств достижимости аффинных по управлению систем с интегральными ограничениями на управление по выходу на малых интервалах времени и асимптотической эквивалентности этих множеств соответствующим множествам достижимости линеаризованных систем.
    \item Доказаны достаточные условия применимости метода линеаризации в задаче локального синтеза для аффинных по управлению систем с интегральными ограничениями.
\end{enumerate}
